Parâmetro e Argumento

Ao criar uma função, você pode definir parâmetros.

Ao execultar uma função. você pode passar argumentos

Funções

// peso e altura são os parâmetros

function imc(peso, altura) {
const imc = peso / (altura ** 2)
b return imc
}

imc(80, 1.80) // 80 e 1.80 são os argumentos

Pode ou não retornar um valor

Quando não definimos o return, ela irá retornar undefined. O códifo interno da função é execultado normalmente, independente de existir valor de return ou n
