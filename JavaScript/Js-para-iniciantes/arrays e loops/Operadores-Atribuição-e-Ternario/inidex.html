Operadores de Atribuição e Ternarios

Atribuição

var x = 5; var y = 10; x += y; // x = x + y (15) x -= y; // x = x - y (-5) x *= y; // x = x * y (50) x /= y; // x = x / y (0.5) x %= y; // x = x % y (0) x **= y; // x = x ** y (9765625)

Ternarios

Abreviação de condicionais com if e else

var idade = 19; var podeBeber = (idade >= 18) ? 'Pode beber' : 'Não pode beber'; console.log(podeBeber) // Pode beber // condição ? true : false

"Geralmente utilizado quando precisamos atribuir um valor para uma variável, dependendo de uma condição"
